Pub-Sub

Ejemplo de uso del patrón Publisher/Subscriber.

Código Js del ejemplo

import pubSub from '../../lib/core.pub-sub'; doPublish : function(){ pubSub.publish(pubSub.TOPICS.NOTIFICATION, 'Botón doPublish'); setTimeout( function(){ pubSub.publish(pubSub.TOPICS.NOTIFICATION, ''); }, 2000); }

Html

<div class="w3-container w3-padding" on-publish="TOPICS.NOTIFICATION:innerHTML">
